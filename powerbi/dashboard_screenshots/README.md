# Dashboard Screenshots

After building the Power BI dashboard following `powerbi_setup_guide.md`,
export each page as a PNG and save here:

- `01_executive_overview.png`
- `02_department_analysis.png`
- `03_compensation_deepdive.png`
- `04_pay_equity.png`
- `05_top_earners.png`

**To export from Power BI Desktop:**
File → Export → Export to PDF, then convert PDF pages to PNG.

Or use snipping tool on each page (Win+Shift+S on Windows).
